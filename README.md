# arduino-bluetooth-controlled-car
DIY Arduino Bluetooth Controlled Car 

materials:

arduino uno, l293d driver, bluetooth hc-05 module, dc motor gear box with wheel, lipo battery 7.4v 50c, led 8mm, dc cables, cardboard.
